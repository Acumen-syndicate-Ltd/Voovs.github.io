Quark_1222

Jan/29'18

Static background image, center-placed text, site icon, dual colour site theme, DNS link, font-style change, and restyling the title are all part of this major update

A static 4k background image was painted onto the canvas. Site load times did suffer by an additional 100ms from the increased painting required. Centralized all the text, using absolute values. Monitors, at 100% zoom, in between 21” and 32” are now optimized for the correct site layout. The Acumen syndicate .Ltd logo has been placed as the site icon. “Acumen syndicate .Ltd” title was given a 3d effect, using a hefty layer of stacked shadows. Site performance slowed down marginally from this. All the fonts on the site were changed to the web safe sans-serif. A dual colour contrast, between #ffffff and #9c2020, was set as the site theme. This is also the first version of the site that was publically available, as connection to the acumensyndicate.ca DNS only happened at this stage
